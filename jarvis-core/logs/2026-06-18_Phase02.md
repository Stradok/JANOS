## Bugs Fixed

### 1. `core/hardware_monitor.py` ŌĆö Zero readings on startup
**Root cause:** `_poll_loop` sampled first then slept, but `asyncio.create_task()` schedules execution asynchronously ŌĆö the task hadn't run by the time callers read `current_load`.  
**Fix:** `start()` now does `self.current_load = await self._sample()` synchronously before creating the polling task. `_poll_loop` now sleeps first, then samples.  
**Verified:** RAM 8.5GB / 15.0GB, CPU 18.3% returned immediately after `await hw.start()`.

### 2. `core/routing.py` ŌĆö `record_agent_result()` arithmetic errors
**Root cause 1:** `fc += 0 if success else fc + 1` ŌĆö doubled the failure counter on every failure instead of incrementing by 1.  
**Root cause 2:** `avg = (score + (sc + fc)) / max(1, sc + fc)` ŌĆö added `score` to the total count before dividing, producing garbage.  
**Fix:** `fc += 0 if success else 1`. Running average now uses: `avg = (old_avg * total_old + score) / total_new` (correct incremental average). Also changed SELECT to include `avg_score` so the old average is available.  
**Verified:** 1 success + 2 failures ŌåÆ `successes=1, failures=2, avg=0.4`.

### 3. `core/llm_client.py` ŌĆö Dead `_client` method conflicts with attribute
**Root cause:** `OllamaProvider` defined both `self._client: httpx.AsyncClient | None = None` (instance attribute in `__init__`) and `async def _client(self)` (class method). The instance attribute shadows the class method, making the method inaccessible. Method was unused ŌĆö all callers use `async with httpx.AsyncClient() as client`.  
**Fix:** Removed the dead `_client()` method entirely.

### 4. `core/reasoning_pipeline.py` ŌĆö `ctx.selected_agent` never set
**Root cause:** Every episode stored `selected_agent = ""` since `WorkflowContext.selected_agent` defaults to `""` and `ReasoningPipeline.run()` never assigned it.  
**Fix:** `ctx.selected_agent = self.executor.name if self.executor else "executor"` added before the execution step.

### 5. `main.py` ŌĆö Strategy refiner created but never started
**Root cause:** `StrategyRefiner` was instantiated and stored in `app.state.strategy_refiner` but `await refiner.start()` was never called. The 24h background loop never ran.  
**Fix:** Added `await app.state.strategy_refiner.start()` immediately after instantiation in the lifespan startup block.

### 6. `agents/crew_agents.py` ŌĆö Agents had no tools wired
**Root cause:** `initialize_agents()` created agents but never called `agent.register_tool()`. Every agent's system prompt said "Available tools: none".  
**Fix:** `initialize_agents()` now registers 7 tools into agents at startup:
- `executor`: web_search, file, browser, system, time, math, weather
- `searcher`: web_search, browser  
- `planner`: web_search  
- `file_operator`: file  
All registrations are wrapped in `try/except` so missing tools degrade gracefully.
