## How to Run

### Quick Demo (no install)
```bash
cd jarvis-core
python demo.py
```

### Full Server
```bash
cd jarvis-core
uvicorn main:app --host 0.0.0.0 --port 8000
```

### Deploy on New PC
```bash
# 1. Copy jarvis-core folder
# 2. Run installer
install.bat
# 3. Launch
python jan_service.py standalone
```
