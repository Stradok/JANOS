Development Workflow — make help for full reference

Setup

make setupFull first-time setup
make installInstall dependencies
make install-devAdd ruff, black, pytest
make install-optionalWake word, face, playwright

Server

make devDev server, auto-reload :8000
make prodProduction, 2 workers
make openOpen browser UI

Chat / CLI

make chatPhase 2-5 full pipeline CLI
make chat-v1Legacy v1 orchestrator CLI
make demoStandalone demo (no server)

Health Checks

make healthGET /health — phases + agents
make statusGET /api/status — hw + mem
make rankingsGET /api/v3/rankings
make scoresGET /api/v5/scores
make auditAll four above in sequence

Testing

make testAll tests
make test-imports17 module import checks
make test-pipeline13 Phase 3-5 integration checks

Lint / Format

make lintruff check
make formatblack in-place
make checklint + format-check (CI gate)

Database

make db-initCreate tables (idempotent)
make db-resetDrop + recreate (with confirm)

Ollama

make ollama-checkStatus + installed models
make ollama-listollama list
make ollama-pull MODEL=…Pull a model

Phase 5 — Learning

make refineStrategy refinement cycle
make generate-tool CAP=…Generate tool stub
make pull-model MODEL=…Pull via JANOS API

Cleanup

make cleanRemove __pycache__ + .pyc
make clean-logsClear daemon log
make clean-memoryReset SQLite DBs
make clean-allCache + logs (keeps DBs)

All targets guard for missing venv/ — run make setup first. Override port with make dev PORT=9000.
